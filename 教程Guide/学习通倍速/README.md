## 二、Windows 客户端
 
1. 下载并打开
 
    [前往下载](https://uopezone.lanzout.com/s/xxtst)

    下载后，解压至桌面

    文件夹里会有两个文件，打开**学习通速通.exe**
 
2. 登陆软件
 
    第一次用就点右下角注册

    注册完登陆即可
 
3. 登陆学习通
 
    使用学习通**扫码登陆**

    点击右上角**回到旧版**

    打开想要快进的视频，**修改下方倍速为 16（最高16），点击左侧 修改倍速 即可**
